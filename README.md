# 📚 EnglishCard

## Приложение для изучения английского языка

Курсовой проект по дисциплине "Базы данных".

Приложение позволяет пользователям изучать английские слова с помощью викторины, добавлять собственные слова, удалять личные слова и отслеживать статистику изучения.

---

# Возможности приложения

## Авторизация пользователя

После запуска приложения пользователь вводит своё имя.

Если пользователь существует в базе данных, выполняется вход.

Если пользователя нет — создаётся новая запись.

---

## Изучение слов

Пользователю предлагается слово на русском языке.

В качестве ответа выводятся 4 варианта перевода на английский язык.

При правильном ответе:

- выводится сообщение об успешном ответе;
- статистика обновляется.

При неправильном ответе:

- предлагается попробовать снова;
- попытка записывается в статистику.

---

## Добавление слов

Пользователь может добавить собственное слово:

- слово на русском языке;
- перевод на английский язык.

Добавленные слова принадлежат только этому пользователю и не отображаются у других пользователей.

---

## Удаление слов

Пользователь может удалить только свои личные слова.

Общие слова из словаря удалить нельзя.

---

## Статистика

В приложении доступна статистика изучения:

- количество правильных ответов;
- количество попыток;
- процент правильных ответов.
